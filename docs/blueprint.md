# **App Name**: Loom Studio

## Core Features:

- Persistent TopBar: TopBar: Persistent navigation and agent context switcher.
- Canvas Zone: Central Canvas Zone: Drag-and-drop grid for visual workflow assembly.
- Palette Panel: Palette Panel: Floating panel with available prompts, agents, and logic blocks.
- Inspector Panel: Inspector Panel: Context-aware editor for node properties.
- Timeline Panel: Timeline Panel: Execution order, token usage, and debug visualizer.
- Agent Hub: Agent Hub Panel: Displays connected agents, their states, permissions, and logs.
- AI Flow Generator: Natural Language Workflow Generation: Use natural language to generate initial prompt workflows. Example: 'Generate a new data cleanup flow'. LLM reasons like a tool to create and place a relevant prompt sequence to achieve the user's goal.

## Style Guidelines:

- Primary color: Light purple (#BB86FC) for a futuristic and immersive feel.
- Background color: Dark gray (#121212) to emphasize content and reduce eye strain, creating a dark mode-forward aesthetic.
- Accent color: Bright pink (#D81B60) to highlight interactive elements and important information.
- Body and headline font: 'Comfortaa' (sans-serif) for headings and user interface elements. 'Lexend' (sans-serif) will be used for the body of any documentation. User specified font(s).
- Minimalist and modern icons that clearly represent different workflow steps and agent actions.
- Responsive grid and motion transitions between canvas zones with snap-to-grid alignment.
- Smooth transitions and subtle animations to provide a fluid and intuitive user experience. Include haptic-like feedback animations.
